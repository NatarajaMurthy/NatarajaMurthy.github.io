Manage personal finances, expenses, budgets and savings goals with ease and precision.

Features

A concise overview of what Money Flow Tracker offers — organized by capability.

1. Comprehensive Transaction Management

  • Income & Expense Tracking: Log amounts, dates, notes and tags for each transaction.
  • Detailed History: Transactions screen with filtering and search (SearchQuery).
  • Account-Based Organization: Manage multiple accounts (Bank, Cash, Cards).

2. Smart Budgeting & Goal Setting

  • Budget Planning: Create and edit budgets by category (AddEditBudgetDialog).
  • Progress Tracking: Visual indicators showing remaining budget.
  • Financial Goals: Track long-term savings and commitments.

3. Advanced Insights & Analytics

  • Visual Trends: Interactive TrendBarCharts for spending over time.
  • Category Breakdown: See which categories consume your budget.
  • Data-Driven Decisions: Use history to identify savings opportunities.

4. Loan & Debt Management

  • Loan Tracking: Manage borrowed/lent money with LoanSettings and DAO support.
  • Repayment Reminders: Keep track of deadlines and commitments.

5. Personalized Category Management

  • Custom Categories: Create, edit, and delete categories via CategoriesScreen.
  • Pre-seeded Data: Default categories included for a fast start.

6. Intelligent Notifications & Reminders

  • Daily Summaries: Receive a daily recap of spending.
  • Smart Alerts: WorkManager-driven alerts for budget limits and goals.
  • Core Alerts: High-priority notifications for critical updates.

7. Cloud Integration & Security

  • Secure Authentication: Firebase Auth and Google Sign-In support.
  • Real-time Sync: Optional Firebase Database sync across devices.
  • Enhanced Security: Google reCAPTCHA and Crashlytics for reliability.

8. Premium Experience & Maintenance

  • In-App Purchases: Google Play Billing for premium features.
  • User Feedback: Built-in FeedbackScreen for suggestions.
  • Seamless Updates: Google Play App Update API integration.
  • Modern UI: Built with Jetpack Compose and Material3 (dark-mode ready).

9. Privacy-First Approach

  • Local Persistence: Room Database as primary on-device storage.
  • Preference Management: DataStore Preferences for settings and onboarding state.
